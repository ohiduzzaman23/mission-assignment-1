## Create a README file to answer the following questions-

1. What is the difference between null and undefined?

- যখন কোনো ভেরিয়েবল ডিক্লেয়ার করা হয় কিন্তু তাকে কোনো মান দেওয়া হয় না, তখন তার ভ্যালু undefined হয়।
- null মানে “কোনো মান নেই” যে সেট করা হয়।

2. What is the use of the map() function in JavaScript? How is it different from forEach()?

- map() ব্যবহার করা হয় একটি array-এর প্রতিটি উপাদানের উপর কাজ করে নতুন একটি array তৈরি করার জন্য।
- forEach() প্রতিটি উপাদানের উপর কাজ করার জন্য ব্যবহার করা হয়, কিন্তু এটি নতুন array রিটার্ন করে না।

3. What is the difference between == and ===?

- == শুধু মান দেখে।
- === value এবং type দুটোই তুলনা করে।

4. What is the significance of async/await in fetching API data?
   -async/await ব্যবহার করা হয় asynchronous কাজ সহজ করার জন্য, বিশেষ করে API থেকে data আনতে।

5. Explain the concept of Scope in JavaScript (Global, Function, Block).

- Scope হলো ভেরিয়েবল কোথায় অ্যাক্সেস করা যাবে (কোন জায়গা থেকে ব্যবহার করা যাবে)।
  - Global সব জায়গা থেকে access করা যায়।
  - Function শুধু ওই ফাংশনের জন্য।
  - Block শুধু ওই {} এর জন্য।
